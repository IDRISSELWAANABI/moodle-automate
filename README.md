# Project Name
Moodle automate
## Overview

This project automates tasks related to Moodle using Selenium. It includes functionalities such as logging into Moodle, downloading course materials, and managing user preferences.

## Requirements

- Python 3
- Selenium 
- requests
- bs4
- tkinter
- winotify
- regex
